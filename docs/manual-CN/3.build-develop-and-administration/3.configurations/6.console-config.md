# 命令行终端设置

属性名                   | 默认值            | 说明 |
------------------------------- | ------------------------ | ----------- |
addr  | "127.0.0.1"	| graphd IP
port  | 0 |	graphd port 
u |	""	| 用于身份验证的用户名
p | ""	| 用于身份验证的密码
enable_history | false | 是否保存历史命令
server_conn_timeout_ms | 1000 | 连接超时时长，单位毫秒
