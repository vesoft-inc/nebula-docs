# SHOW USERS

The `SHOW USERS` statement shows the user information.

## Role requirement

Only the `root` user who has the `GOD` role can use the `SHOW USERS` statement.

## Syntax

```ngql
SHOW USERS;
```

## Example

```ngql
nebula> SHOW USERS;
+---------+
| Account |
+---------+
| "root"  |
| "user1" |
+---------+
```
