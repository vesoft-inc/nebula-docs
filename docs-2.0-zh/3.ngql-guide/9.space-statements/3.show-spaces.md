# SHOW SPACES

`SHOW SPACES`语句可以列出{{nebula.name}}示例中的所有图空间。

## 语法

```ngql
SHOW SPACES;
```

## 示例

```ngql
nebula> SHOW SPACES;
+--------------------+
| Name               |
+--------------------+
| "cba"              |
| "basketballplayer" |
+--------------------+
```

创建图空间请参见 [CREATE SPACE](1.create-space.md)。
