# SHOW HOSTS

`SHOW HOSTS`语句可以显示集群信息，包括端口、状态、leader、分片、版本等信息，或者指定显示 Graph、Storage、Meta 服务主机信息。

## 语法

```ngql
SHOW HOSTS [GRAPH | STORAGE | META];
```

!!! note

    对于使用源码安装的{{nebula.name}}，执行添加了服务名的命令后，输出的信息中不显示版本信息。
