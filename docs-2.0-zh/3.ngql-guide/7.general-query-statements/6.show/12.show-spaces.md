# SHOW SPACES

`SHOW SPACES`语句显示现存的图空间。

如何创建图空间，请参见 [CREATE SPACE](./../../9.space-statements/1.create-space.md)。

## 语法

```ngql
SHOW SPACES;
```

## 示例

```ngql
nebula> SHOW SPACES;
+---------------------+
| Name                |
+---------------------+
| "docs"              |
| "basketballplayer"  |
+---------------------+
```
