# SHOW USERS

`SHOW USERS`语句显示用户信息。

## 角色要求

只有`GOD`角色的用户（即`root`）才能执行`SHOW USERS`语句。

## 语法

```ngql
SHOW USERS;
```

## 示例

```ngql
nebula> SHOW USERS;
+---------+-----------------+
| Account | IP Whitelist    |
+---------+-----------------+
| "root"  | ""              |
| "user1" | ""              |
| "user2" | "192.168.10.10" |
+---------+-----------------+
```
