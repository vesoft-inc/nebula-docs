# 算术运算符

 {{nebula.name}} 支持的算术运算符如下。

| 符号  | 说明 |
|:----|:----:|
| `+`   | 加法   |
| `-`   | 减法   |
| `*`   | 乘法   |
| `/`  | 除法   |
| `%`   | 取模   |
| `-`   | 负数符号 |

## 示例

```ngql
nebula> RETURN 1+2 AS result;
+--------+
| result |
+--------+
| 3      |
+--------+

nebula> RETURN -10+5 AS result;
+--------+
| result |
+--------+
| -5     |
+--------+

nebula> RETURN (3*8)%5 AS result;
+--------+
| result |
+--------+
| 4      |
+--------+

```
