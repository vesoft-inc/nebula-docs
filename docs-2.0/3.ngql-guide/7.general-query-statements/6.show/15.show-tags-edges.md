# SHOW TAGS/EDGES

The `SHOW TAGS` or `SHOW EDGES` statement shows all tags or edge types in the current graph space.

## Syntax

```ngql
SHOW {TAGS | EDGES}
```

## Examples

Show tags:

```ngql
nebula> SHOW TAGS;
+----------+
| Name     |
+----------+
| "player" |
+----------+
| "star"   |
+----------+
| "team"   |
+----------+
Got 3 rows (time spent 1461/2114 us)
```

Show edge typesï¼š

```ngql
nebula> SHOW EDGES;
+---------+
| Name    |
+---------+
| "like"  |
+---------+
| "serve" |
+---------+
Got 2 rows (time spent 1039/1687 us)
```
