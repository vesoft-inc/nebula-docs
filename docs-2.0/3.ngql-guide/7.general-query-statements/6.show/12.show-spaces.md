# SHOW SPACES

The `SHOW SPACES` statement shows the graph spaces in Nebula Graph.

For how to create a graph space, see [CREATE SPACE](./../../9.space-statements.md/1.create-space.md).

## Syntax

```ngql
SHOW SPACES
```

## Example

```ngql
nebula> SHOW SPACES;
+--------+
| Name   |
+--------+
| "docs" |
+--------+
| "nba"  |
+--------+
Got 2 rows (time spent 968/1893 us)
```
