# SHOW SPACES

`SHOW SPACES` lists all the graph spaces in the NebulaGraph examples.

## Syntax

```ngql
SHOW SPACES;
```

## Example

```ngql
nebula> SHOW SPACES;
+--------------------+
| Name               |
+--------------------+
| "cba"              |
| "basketballplayer" |
+--------------------+
```

To create graph spaces, see [CREATE SPACE](1.create-space.md).
