# Show INDEXES

```ngql
SHOW {TAG | EDGE} INDEXES
```

Use `SHOW INDEXES` to list the defined tag or edg-type indexes names.

## Example

```ngql
nebula> SHOW TAG INDEXES;
+------------------+
| Names            |
+------------------+
| "fix"            |
+------------------+
| "player_index_0" |
+------------------+
| "player_index_1" |
+------------------+
| "var"            |
+------------------+

nebula> SHOW EDGE INDEXES;
+------------------+
| Names            |
+------------------+
| "follow_index_0" |
+------------------+
```
