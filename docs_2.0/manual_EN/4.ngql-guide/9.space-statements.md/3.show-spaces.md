# SHOW SPACES

```ngql
SHOW SPACES
```

The `SHOW SPACES` statement lists the all the graph spaces in a Nebula Graph instance.

For example:

```ngql
nebula> SHOW SPACES;
+--------+
| Name   |
+--------+
| "nba"  |
+--------+
```

To create graph spaces, see [Create Space document](1.create-space.md).
